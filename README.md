# Formulario de Registro en Python con PyQt5

## Descripción
Formulario de registro de usuarios desarrollado en **Python 3.10** usando **PyQt5**. Permite capturar datos personales como **nombre, apellidos, correo electrónico y celular**, con validaciones de formato y restricciones, demostrando habilidades en el manejo de **interfaces gráficas, estilos personalizados y eventos interactivos**.

## Características
- Interfaz gráfica amigable y moderna.
- Validación de campos:
  - Solo letras en nombre y apellidos.
  - Formato válido de correo electrónico.
  - Celular de 10 dígitos numéricos.
- Botones interactivos:
  - **Registrar**: verifica la información antes de guardar.
  - **Cancelar**: limpia todos los campos y notifica la acción.
- Estilos CSS personalizados para etiquetas, cajas de texto y botones.

## Tecnologías y Librerías
- **Python 3.10**
- **PyQt5**
- **Qt Designer** (opcional si se usa archivo `.ui`)
- **re** (expresiones regulares para validar correo y nombre)
